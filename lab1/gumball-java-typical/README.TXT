PROJECT TITLE: gumball-java-typical
PURPOSE OF PROJECT: The purpose of this project is to instantiate gumball machines that can support the requirements of three different gumball machine models. The first machine is one that takes only $0.25 in quarters, the second machine takes $0.50 only in quarters, and the third machine takes $0.50 in various types of coins (penny, nickel, dime, quarter). 
VERSION or DATE: 02/08/19
HOW TO START THIS PROJECT: Run the Main class.
AUTHORS: Krishnamurthy Thyagarajan
USER INSTRUCTIONS: Open the project in BlueJ and right click the Main class. Run the static method and look at the console for output.
